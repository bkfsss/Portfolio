**Divider** — a hairline rule for separating stacked sections. Optionally labelled.

```jsx
<Divider />
<Divider label="2024" />
<Divider vertical />
```

Use plain for section breaks, `label` to mark a year or phase between entries, `vertical` inside flex rows. Uses the faint `--line-2` hairline so it whispers rather than shouts.
