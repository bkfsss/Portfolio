**Badge** — a small mono chip for tech tags, statuses, and metadata. Square corners, tiny uppercase-friendly mono type.

```jsx
<Badge>react</Badge>
<Badge variant="solid">new</Badge>
<Badge variant="muted">archived</Badge>
```

Use `outline` (default) for tech stacks, `solid` for emphasis (e.g. "new"), `muted` for de-emphasized states. Group several with a flex `gap` row under a project title.
