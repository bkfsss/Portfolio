Drop your gallery photos in THIS folder.

Name them: 01.jpg, 02.jpg, 03.jpg, 04.jpg
(.png works too — just change the extension in photography.html's FRAMES list.)

- Each filename maps to a frame in the gallery, in order.
- To add MORE photos: add another entry to the FRAMES array in
  ui_kits/portfolio/photography.html (copy a line, bump the number).
- Edit each frame's `cap` (caption) and `meta` (lens/settings) there too.
- If a file is missing, that frame shows the crimson placeholder instead.
